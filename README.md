# Sysrandom

Secure random number generation for Ruby using system RNG facilities (e.g. `/dev/urandom`, `getrandom()`)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sysrandom'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sysrandom
